🔧 CommonUtils修复测试

📋 问题描述

work-circle.js中出现了以下错误：

错误1: work-circle.js:626 Uncaught ReferenceError: commonUtils is not defined
错误2: work-circle.js:312 Uncaught ReferenceError: commonUtils is not defined

原因分析：

  • work-circle.js在页面加载时立即执行，但commonUtils可能还未完全初始化
  • 脚本加载顺序正确，但存在时序问题
  • 需要添加安全检查机制

🛠️ 修复方案

1. 添加安全包装函数：

function safeCommonUtils() { if (typeof window.commonUtils !== 'undefined') { return window.commonUtils; } // 返回备用函数 return { showToast: function(message, type) { ... }, navigateTo: function(url) { ... }, // 其他备用函数... }; }

2. 替换所有commonUtils调用：

// 修改前 commonUtils.showToast('消息', 'info'); // 修改后 safeCommonUtils().showToast('消息', 'info');
