🔧 问题点击跳转修复测试

测试专家问答页面中问题列表项的点击跳转功能修复

1. 问题描述

发现的问题：
  • 热门问题区域点击正常：跳转到 question-detail.html?id=q002
  • 问题列表中点击异常：跳转到 question-detail.html?id=undefined
问题原因：
  • 问题列表项缺少 data-question-id 属性
  • JavaScript的 initQuestionList 函数依赖 data-question-id 属性
  • 热门问题使用 onclick 直接调用 viewQuestion 函数

2. 修复方案对比

问题项 修复前 修复后
热门问题1 (q001)
onclick="viewQuestion('q001')"
data-question-id="q001"
热门问题2 (q002)
onclick="viewQuestion('q002')"
data-question-id="q002"
热门问题3 (q003)
onclick="viewQuestion('q003')"
data-question-id="q003"
原有问题 (q004-q006)
onclick="viewQuestion('q00X')"
data-question-id="q00X"

3. 技术实现说明

JavaScript事件处理机制：
// initQuestionList 函数中的事件绑定 item.addEventListener('click', function() { const questionId = this.dataset.questionId; // 读取 data-question-id openQuestionDetail(questionId); });
热门问题区域保持不变：
  • 热门问题使用 onclick="viewQuestion('q00X')" 方式正常工作
  • 问题列表使用 data-question-id 属性配合事件监听器
  • 两种方式都能正确跳转到问题详情页面
